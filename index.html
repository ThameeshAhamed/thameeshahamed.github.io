My Projects

Wireless Smart Planting

This project is a smart agricultural system developed to improve farming efficiency through automation. It uses the ESP32 microcontroller as the core component, enabling wireless control and automated operations. The system includes three types of servo motors for vehicle movement and robotic arm functions, allowing it to perform tasks such as navigation, planting, and watering without human intervention. Designed specifically for agricultural fields, the project helps in reducing labor costs, saving time, and enhancing overall productivity. It demonstrates how technology can be effectively applied to support modern, sustainable farming practices.

KCET Department Website

This project involved the design and development of a responsive and interactive website for the KCET college department. The website was built using HTML, CSS, and JavaScript, focusing on clean UI design, intuitive navigation, and student-centered features. It includes multiple sections such as Home, Student Achievements, Faculty Details, and College Highlights. One of the key features is a dedicated student portal that allows users to view exam timetables, access recent and upcoming events, and navigate directly to external university portals. The site is optimized for both desktop and mobile viewing, ensuring a seamless experience for all users. This project demonstrates strong front-end development skills, an understanding of user experience (UX) principles, and the ability to design solutions tailored to educational institutions.
